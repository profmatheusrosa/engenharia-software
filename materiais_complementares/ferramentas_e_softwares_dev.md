# Ferramentas e Softwares Recomendados

## Editores e IDEs
- **VS Code:** Leve, extensível, padrão de mercado para Web/JS/TS.
- **IntelliJ IDEA:** A melhor IDE para Java/Kotlin.
- **PyCharm:** A melhor IDE para Python.

## Banco de Dados
- **DBeaver:** Cliente universal gratuito para qualquer banco de dados.
- **Postman / Insomnia:** Para testar APIs REST.

## Design e Modelagem
- **Draw.io (Diagrams.net):** Gratuito, web e desktop. Ótimo para UML e C4.
- **Excalidraw:** Estilo "quadro branco", visual de rascunho. Excelente para brainstorming.
- **Figma:** Padrão para prototipagem de UI/UX.

## Produtividade e Gestão
- **Notion:** Documentação, wikis e anotações.
- **Trello:** Kanban simples.
- **Jira:** Gestão ágil robusta.

## Terminal
- **Windows Terminal:** Essencial se usa Windows.
- **Oh My Zsh:** Turbina seu terminal (Linux/Mac/WSL).
