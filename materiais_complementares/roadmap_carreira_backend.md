# Roadmap de Carreira: Engenheiro de Software (Backend Focus)

Este guia sugere uma trilha de estudos para quem deseja se especializar, baseada no conteúdo deste curso.

## Nível 1: A Base (O Curso Cobre Aqui)
- [x] Lógica de Programação
- [x] Git e Versionamento
- [x] HTTP e APIs REST
- [x] Banco de Dados Relacional (SQL)
- [x] Testes Unitários

## Nível 2: Profundidade
- [ ] **Linguagem:** Domine profundamente uma (Java, C#, Node.js, Go). Conheça a JVM ou o Runtime.
- [ ] **Docker:** Aprenda a criar containers para suas aplicações.
- [ ] **Banco de Dados:** Aprofunde em modelagem, índices e otimização de queries.
- [ ] **Segurança:** OWASP Top 10, Autenticação (JWT, OAuth2).

## Nível 3: Arquitetura e Escala
- [ ] **Cloud:** AWS, Azure ou GCP. (EC2, S3, RDS, Lambda).
- [ ] **NoSQL:** Redis (Cache), MongoDB (Documentos), ElasticSearch (Busca).
- [ ] **Mensageria:** RabbitMQ, Kafka. (Comunicação assíncrona).
- [ ] **Observabilidade:** Logs (ELK), Métricas (Prometheus/Grafana), Tracing.

## Nível 4: Liderança Técnica (Staff+)
- [ ] **System Design:** Projetar sistemas complexos e distribuídos.
- [ ] **Soft Skills:** Mentoria, Negociação, Comunicação Técnica.
- [ ] **Custos:** Otimização de infraestrutura (FinOps).
