# Engenharia de Software Moderna

Curso completo abordando desde os fundamentos da Engenharia de Software até arquiteturas modernas e DevOps, com foco em práticas de mercado.

## Visão Geral do Curso

- **Carga Horária Total:** 40 horas
- **Duração por Aula:** 1 hora (sugerida)

## Pré-requisitos e Ambiente

- Editor de Código (VS Code recomendado).
- Git instalado.
- Node.js (para ferramentas modernas e exemplos JS/TS).
- Conhecimentos básicos de programação.

## Integridade Acadêmica

> "O aprendizado é uma jornada pessoal. Copiar a solução de outros priva você da oportunidade de crescer."

Encorajamos o uso responsável de ferramentas de IA (como ChatGPT, Claude, Gemini) como auxiliares de estudo (ex: "explique este conceito", "gere exemplos de dados"), mas desencorajamos fortemente o uso para gerar soluções completas sem compreensão. O valor deste curso está na prática.
