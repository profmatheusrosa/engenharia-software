# Exercícios - Módulo 1: Fundamentos Modernos

## Exercício Prático 1: Configurando o Ambiente e Gestão Visual

**Objetivo:** Familiarizar-se com ferramentas de gestão ágil e configuração de ambiente de desenvolvimento.

**Cenário:** Você acaba de entrar em uma nova equipe de desenvolvimento que utiliza Scrum. Seu primeiro trabalho é configurar seu ambiente e entender o fluxo de trabalho no Trello (ou GitHub Projects).

**Passos:**
1. Crie uma conta no Trello (ou use o GitHub Projects).
2. Crie um quadro chamado "Meu Primeiro Projeto Ágil".
3. Crie as colunas: `Backlog`, `To Do` (Sprint Backlog), `Doing`, `Code Review`, `Done`.
4. Adicione 3 cartões na coluna `Backlog` representando tarefas fictícias (ex: "Configurar IDE", "Ler documentação do projeto", "Criar Hello World").
5. Simule o ciclo de vida de uma tarefa: mova de `Backlog` para `To Do`, depois `Doing` e finalmente `Done`.

**Entregável:** Tire um print do seu quadro com os cartões na coluna `Done` e salve como `exercicio_01_seunome.png`.

## Exercício Prático 2: Análise de Qualidade

**Objetivo:** Identificar atributos de qualidade em um cenário real.

**Cenário:** Um cliente pede um aplicativo de e-commerce que deve suportar a Black Friday sem cair.

**Passos:**
1. Crie um arquivo de texto.
2. Liste 3 Requisitos Não Funcionais críticos para esse cenário baseados na ISO 25010 (vistos na teoria).
3. Para cada requisito, escreva uma breve justificativa de por que ele é importante para o e-commerce na Black Friday.
